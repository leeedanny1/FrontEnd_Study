document 객체
  : document객체는 웹 페이지에 존재하는 HTML요소에 접근하고자 할 때 사용하는 객체입니다.

  getElementsByTagname
    : 해당 태그의 이름으로 요소를 모두 선택
  getElementByID
    : 해당 아이디의 요소를 선택
  getElementsByClassName
    : 해당 클래스에 속한 요소를 모두 선택
  getElementsByName
    : 해당 name 속성값을 가지는 요소를 모두 선택
  querySelectorAll
    : 해당 선택자로 선택되는 요소를 모두 선택

  문서 객체 모델(Document Object Model -> DOM)
    : HTML문서나 XML문서등을 접근하기 위한 일종의 인터페이스입니다.
    이 객체 모델은 문서내의 모든 요소를 정의하고,
    각각의 요소에 접근하는 방법을 제공합니다.
